# قوالب Obsidian (Templater) لكتابة الملاحظات

قوالب تُولِّد سطر الميتاداتا بالصيغة التي يتوقّعها بناء الموقع تمامًا، فتظهر
شارة النوع والختمة، والخطّ الزمني، وروابط الوسوم، ويمرّ الفحص — كلّه تلقائيًّا.

> هذا المجلد خارج `surahs/`، فلا يصبح صفحةً في الموقع.
> ملاحظة: صفحات **تدبر السور** تُولَّد آليًّا (Gemini)، فالقوالب هنا للملاحظات
> اليدوية في `verses/` و`fajr/` فقط.

## محتوى المجلد

```
obsidian-templates/
├── verse-note.md          ← قالب ملاحظة تدبّر آية (لـ surahs/verses/)
├── fajr-note.md           ← قالب فائدة فجر موضوعية (لـ surahs/fajr/)
├── scripts/
│   ├── surah_name.js      ← رقم السورة → اسمها (مُولَّد من surahs.py)
│   └── get_ayah.js        ← نصّ الآية العثماني محليًّا
└── data/
    └── quran-uthmani.json ← 6236 آية بخريطة "سورة:آية" → النصّ (دون اتصال)
```

## الإعداد (مرّة واحدة)

الـ vault هو مجلد مشروع `quran-fagr`. في Obsidian:

1. **Settings → Community plugins → Templater** (مُثبَّت لديك).
2. **Templater → Template folder location:** اضبطه على `obsidian-templates`.
3. **Templater → User Scripts → Script files folder location:** اضبطه على
   `obsidian-templates/scripts`.
4. (اختياري) عيّن اختصارًا للأمر **Templater: Open insert template modal**.

## الاستخدام

1. افتح الصفحة الهدف، مثلًا `surahs/verses/002-البقرة.md`
   (أو ملف باب في `surahs/fajr/`).
2. ضع المؤشّر حيث تريد الملاحظة — **أعلى القائمة** (الأحدث أولًا).
3. شغّل **Templater: Open insert template modal** واختر `verse-note`
   (أو `fajr-note`).
4. أجِب عن الأسئلة: رقم السورة → يُملأ الاسم، رقم الآية → يُجلَب نصّها،
   ثم الوسوم. يُدرَج القالب بالصيغة الصحيحة ويبقى المؤشّر لتكتب التأمّل.

### مثال للناتج (verse-note)

```markdown
## آية 255:
> "ٱللَّهُ لَآ إِلَٰهَ إِلَّا هُوَ ٱلْحَىُّ ٱلْقَيُّومُ ..."

*2026-06-05 · #توحيد · #آية-الكرسي · [البقرة:255](../quran/002-البقرة.md)*
```

## تحديث/صيانة

- بيانات الآيات ثابتة؛ لا تحتاج تحديثًا. لإعادة توليدها:
  نزّل `quran-uthmani` من alquran.cloud وأعد تشكيله إلى خريطة `"سورة:آية"`.
- إن أُعيدت تسمية سورة في `scripts/surahs.py`، أعد توليد `surah_name.js` منه.
- مسار ملفّ البيانات داخل `get_ayah.js` نسبيٌّ من جذر الـ vault، فأبقِ
  المجلد في مكانه.
