# مشاركات حلقة الفجر

ملاحظاتٌ يومية مقسَّمة إلى أبواب: لغة، بلاغة، تجويد، أسئلة وأجوبة، فقه، وغيرها.
أضف بابًا جديدًا بإنشاء ملف `.md` جديد داخل `surahs/fajr/` — يظهر في الشريط الجانبي تلقائيًا.

## كيف تضيف فائدة

افتح ملف الباب المناسب (مثلًا `surahs/fajr/فوائد-لغوية.md`) وأضف هذا القالب أعلى الملف
(أحدث ملاحظة أولًا):

```markdown
## عنوان الفائدة
*2026-06-02 · #وسم · [البقرة:255](../quran/002-البقرة.md)*

> "اقتباس آية إن وُجد"

نصّ الفائدة… استخدم **التشديد** و*اللون* و==التظليل==.
```

- يظهر العنوان الفرعي تلقائيًا في الشريط الجانبي لصفحة الباب (TOC).
- زر **🔗** بجوار العنوان يعطيك رابطًا مباشرًا للمشاركة.
- التاريخ في سطر الميتاداتا يُلحق الملاحظة بالختمة المناسبة في تبويب
  [ختمات](../khatmas/) تلقائيًا، ويُضيفها لصفحة [الجديد](../whats-new.md).
- الرابط للسورة (`[البقرة:255](../quran/002-البقرة.md)`) يجعلها تظهر في
  [فهرس حسب السورة](../notes-by-surah.md).

## التحرير من الهاتف عبر GitHub

1. افتح المستودع: https://github.com/JohanLiebert66/quran-fagr
2. تصفّح إلى الملف المطلوب (مثلًا `surahs/fajr/فوائد-لغوية.md`).
3. اضغط ✏️ (Edit this file) واكتب فائدتك بنفس القالب أعلاه.
4. اضغط **Commit changes**.
5. خلال ~ دقيقتين، GitHub Actions يبني الموقع تلقائيًا ويُحدّث
   [https://johanliebert66.github.io/quran-fagr/](https://johanliebert66.github.io/quran-fagr/).
